Country Structures Statistics

Role: Data & GIS Developer
Tech: Python, GeoJSON, Pandas, Matplotlib, Excel
Focus: Automated building footprint statistics

Overview

This project analyses building footprint data for countries or regions and automatically generates statistics on building sizes. It reads GeoJSON files containing structures, computes building areas, groups them into size intervals, and exports both Excel tables and PDF histograms.

The whole workflow is designed to be batch-friendly: drop new GeoJSON files into the input folder, run the script, and all outputs and summaries are generated automatically.

Problem & Motivation

For comparing urban form across regions, it’s useful to understand the distribution of building sizes (e.g. many small structures vs. fewer large ones). Doing this manually for each region is slow and error-prone, so I built a reusable tool that can process multiple countries or areas in one run.

Data & Inputs

  • GeoJSON building footprints stored under files/input/.
  • metadata.csv providing population, land area, and labels for each region.
  • Configurable settings in config.py for input/output paths.

Key Features

1. GeoJSON Input Processing

  • Automatically scans files/input/ for *.geojson files.
  • Extracts areas for each building footprint.

2. Area Binning

  • Groups building areas from 0–900 m² into fixed-size bins (default 5 m²).
  • Handles buildings larger than 900 m² in an overflow bucket.

3. Excel Export

  • Writes an interval vs. count table for each region into files/output/.
  • Makes the data easy to use in spreadsheets or further analysis.

4. PDF Histogram Generation

  • Creates clear bar charts with labelled axes and counts.
  • Optionally includes population and land area summaries.
  • Uses metadata.csv to match each GeoJSON file with the correct stats.

5. Command-Line Batch Mode

  • Run everything with a single command: python main.py.
  • Designed to be triggered from VS Code or the terminal.

How I Implemented It

  1. Defined configuration in config.py for input/output folders.
  2. Used GeoJSON parsing and geometry libraries to compute building areas.
  3. Implemented an area-binning function so the bin size is easy to change.
  4. Used Pandas to build interval tables and export them to Excel.
  5. Used Matplotlib to create histograms and save them as PDFs.

What I Learned

  • How to design a reproducible batch-processing workflow for spatial data.
  • How to connect geometric operations with statistical summaries and visualisations.
  • Practical experience handling metadata joins and matching filenames to reference tables.
